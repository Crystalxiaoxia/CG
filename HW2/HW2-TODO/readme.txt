=============================
HW2：几何建模与场景漫游
=============================

一、实验目的
------------
本实验旨在通过 WebGL 和 Three.js 技术，实现几何建模与三维场景漫游。  
学生需掌握三维图形的建模、PVM 矩阵变换、投影方式切换以及交互控制等核心内容。

实验分为两个部分：
1. 基于 WebGL 的几何建模与投影控制；
2. 基于 Three.js 的复杂场景建模（太阳系系统）与交互观察。

-------------------------------------------------------------

二、主要功能
------------
基本要求（90分）：
1. 草帽模型建模与渲染；
2. 键盘交互控制：
   - Z / C ：模型整体缩放；
   - W / S / A / D ：相机绕 X、Y 轴旋转；
   - P ：正投影与透视投影切换；
   - M / N ：调整视角大小；
   - R / T ：开启 / 关闭后向面剔除；
   - B / V ：开启 / 关闭深度缓存消隐；
3. 鼠标拖动控制相机视角；
4. 空格键：复位到初始状态。

附加项（10分）：
1. Three.js 实现太阳系动态模型（太阳、地球、月球的自转与公转）；
2. 支持鼠标拖拽、滚轮缩放、键盘控制投影模式、光照与材质效果；
3. WebGL 与 Three.js 双模式切换；
4. 页面自适应与界面美化。

-------------------------------------------------------------

三、运行说明
------------
1. 运行环境：
   - 浏览器：Chrome / Firefox / Edge（需支持 WebGL）
   - 推荐编辑器：VS Code
   - 本地服务器：使用 Live Server 或 http-server 运行项目

2. 运行步骤：
   （1）将项目文件放置在同一目录下；
   （2）使用 VS Code 打开项目文件夹；
   （3）右键运行 “3d-wandering.html”；
   （4）通过下拉菜单选择：
        - cube（立方体）
        - sphere（球体）
        - hat（墨西哥草帽）
        - solar（太阳系模型）

3. 操作方法：
   - 鼠标拖动：旋转视角
   - 鼠标滚轮：缩放
   - 键盘：
       W / S ：绕 X 轴旋转
       A / D ：绕 Y 轴旋转
       Z / C ：整体缩放
       P ：切换投影模式
       M / N ：调整视角大小
       R / T ：开启 / 关闭面剔除
       B / V ：开启 / 关闭深度缓存消隐
       空格 ：一键复位

五、实现思路简述
----------------
1. WebGL 部分：
   - 使用 PVM 矩阵完成坐标变换；
   - 通过 JavaScript 动态控制相机与投影参数；
   - 在片元着色器中渲染模型颜色；
   - 响应键盘与鼠标事件实现交互控制。

2. Three.js 部分：
   - 利用 SphereGeometry 建立太阳、地球、月球模型；
   - 加载 NASA 官方纹理；
   - 使用 Clock 与动画循环实现自转与公转；
   - 结合 OrbitControls 实现鼠标漫游；
   - 扩展键盘控制实现缩放、投影、深度与剔除控制；
   - 空格键实现一键复位。
