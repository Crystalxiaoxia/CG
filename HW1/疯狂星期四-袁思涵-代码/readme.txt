使用说明（README）
1.打开项目
在 VS Code 等开发环境中打开 “疯狂星期四-袁思涵-代码” 文件夹。
右键点击 gasket2D.html 文件，选择 “Open in Live Server”，即可在浏览器中运行 2D Sierpinski Gasket 页面。
2.调整递归层数
使用页面上的 滑动条，或按下 键盘上下方向键，可以动态改变图形的递归次数，从而观察分形结构的变化。
3.控制旋转
点击 “Start/Stop Rotation” 按钮，可以切换图形的旋转状态，实现开始或停止旋转的效果。
4.调整旋转速度
点击 “speadUP” 按钮可加快旋转速度；
点击 “speadDOWN” 按钮可减慢旋转速度。
5.改变旋转中心
在画布区域内任意点击一个新位置，程序会将该点作为新的图形中心，并以此重新绘制分形图形。
6.查看实时信息
在 “speadDOWN” 按钮旁，会实时显示当前的 帧率（FPS） 以及图形的 顶点数量，方便观察性能变化。
